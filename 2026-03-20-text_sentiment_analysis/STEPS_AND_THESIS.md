# Text Sentiment Analysis using Machine Learning
## Introduction
This project uses a machine learning approach to classify text as positive or negative based on its sentiment. The project uses a labeled dataset to train a model and then evaluate its performance on a test set.
## Dependencies
* numpy
* pandas
* scikit-learn
* nltk
## How to Run
1. Install the dependencies using pip install numpy pandas scikit-learn nltk
2. Run the main.py file using python main.py
