# Facial Emotion Recognition Using Deep Learning
## Introduction
This project aims to develop a deep learning model that can recognize human emotions from facial expressions. The model will be trained on a dataset of images with different emotions and will be able to predict the emotion of a given image.
## Dependencies
* numpy
* pandas
* scikit-learn
* tensorflow
* keras
* opencv-python
## Usage
1. Clone the repository.
2. Install the dependencies using pip.
3. Run the main.py file to train and evaluate the model.
