# Text Sentiment Analysis using Machine Learning
## Introduction
This project aims to classify text as positive, negative, or neutral using machine learning algorithms.
## Dependencies
* numpy
* pandas
* scikit-learn
* nltk
## Usage
1. Install the dependencies using pip.
2. Run the main.py script to train and evaluate the model.
## Dataset
The dataset used in this project is a CSV file containing text data and their corresponding labels.
## Model
The model used in this project is a Naive Bayes classifier with TF-IDF vectorization.
